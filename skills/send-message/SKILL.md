# 채팅방에 메시지 보내기

채팅방: **$ARGUMENTS**

## 절차

1. 내 이름을 확인한다 (role.md 또는 현재 프로젝트 디렉토리에서 판단)

2. chatrooms 디렉토리를 찾는다. 현재 프로젝트의 상위에 `chatrooms/` 폴더가 있어야 한다.

3. `chatrooms/$ARGUMENTS/` 디렉토리의 마지막 파일명을 확인하여 다음 순번을 결정한다.
   - 파일명 규칙: `{날짜}_{순번}.md`
   - 같은 날짜면 순번 +1, 새 날짜면 001부터
   - 날짜는 UTC 기준

4. 사용자에게 메시지 내용을 물어본다 (또는 이미 대화 맥락에서 보낼 내용이 명확하면 바로 작성)

5. 메시지 파일을 생성한다:
```markdown
---
from: {내 이름 소문자}
to: {상대 에이전트 이름 소문자 또는 all}
time: {YYYY-MM-DD HH:MM UTC}
subject: {제목}
mentions: [{멘션할 에이전트 목록}]
---

{내용에서 @이름으로 멘션}
```
   - `mentions`는 선택사항. 단체방(general 등)에서 특정 에이전트에게 알릴 때 사용
   - 1:1 채팅방에서는 생략

6. 자신의 `.read-status/{내이름}.json`을 업데이트하여 방금 보낸 파일을 마지막 읽은 것으로 기록한다.

7. 사용자에게 전송 완료를 알린다.
