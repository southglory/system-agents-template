# 내 멘션 확인

## 절차

1. 내 이름을 확인한다 (role.md 또는 현재 프로젝트 디렉토리에서 판단)

2. chatrooms 디렉토리를 찾는다. 현재 프로젝트의 상위에 `chatrooms/` 폴더가 있어야 한다.

3. chatrooms 아래의 모든 채팅방 디렉토리를 조회한다.

4. `.read-status/{내이름}.json`을 읽어서 각 채팅방의 마지막 읽은 파일명을 확인한다.

5. 각 채팅방에서 마지막 읽은 파일 이후의 새 메시지를 확인한다.
   - read-status에 해당 채팅방 키가 없으면 모든 메시지를 확인한다.

6. 새 메시지 중 다음 조건에 해당하는 것만 사용자에게 보여준다:
   - frontmatter `mentions`에 내 이름이 포함된 메시지
   - frontmatter `to`가 내 이름이거나 `all`인 메시지 (1:1 채팅방)

7. 결과 형식:
   ```
   📢 [채팅방이름] @나를 멘션 — {from}이(가) "{subject}" (시간)
   ```
   멘션된 메시지가 없으면 "멘션 없음"으로 표시.

8. 사용자가 읽기를 원하는 메시지가 있으면 해당 메시지를 읽고 read-status를 업데이트한다.
